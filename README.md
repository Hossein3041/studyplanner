# studyplanner

Webbasierte Anwendung zur strukturierten Organisation des Studienalltags,
insbesondere zur Verwaltung von Modulen, Aufgaben, Lernphasen und Zeiteinheiten.
Der Fokus liegt auf der Entwicklung einer klar strukturierten Single Page Application
mit clientseitiger Zustandsverwaltung und modularer Frontend-Architektur.
Im Backend werden RESTful Webservices mit Java (Jakarta Servlets) umgesetzt,
inklusive Sitzungsverwaltung, Authentifizierung und Anbindung an eine relationale
Datenbank (MariaDB).
Das Frontend basiert auf HTML, CSS und JavaScript.
Das Projekt wird versionskontrolliert mit Git entwickelt und befindet sich aktuell
noch in aktiver Weiterentwicklung.
